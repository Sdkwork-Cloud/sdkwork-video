# Tools Directory

## Purpose
Developer, validation, generation, migration, and operator tools that are not shipped as app runtime code.

## Owner
Developer experience teams and tool maintainers.

## Allowed Content
- Validation tools
- Code generators
- Migration scripts
- Operator utilities
- Developer CLI tools

## Forbidden Content
- Application runtime code
- Generated SDK output
- API contracts
- Deployment descriptors

## Related Specs
- `CODE_STYLE_SPEC.md`
- `TEST_SPEC.md`
- `DOCUMENTATION_SPEC.md`

## Verification
- Tools are documented with usage instructions
- Tools are deterministic and safe to run
- Tools do not store secrets or credentials
