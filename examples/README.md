# Examples Directory

## Purpose
Runnable examples, integration examples, sample configs, and SDK/API usage examples.

## Owner
Documentation teams and SDK developers.

## Allowed Content
- Runnable example code
- Integration examples
- Sample configurations
- SDK/API usage examples

## Forbidden Content
- Production application code
- Generated SDK output
- API contracts
- Secrets or credentials

## Related Specs
- `DOCUMENTATION_SPEC.md`
- `SDK_SPEC.md`
- `API_SPEC.md`

## Verification
- Examples are runnable and documented
- Examples use safe sample data
- Examples do not contain real credentials
